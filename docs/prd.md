# IQ Testing and Training Website需求文档

## 1. 网站概述

### 1.1 网站名称

IQ Train

### 1.2 网站描述

专业的智商测试和大脑训练平台，提供权威的IQ评估服务和个性化的认知训练课程，帮助用户了解自身智力水平并持续提升思维能力。

## 2. 功能需求

### 2.1 核心功能

#### 2.1.1 IQ测试模块

- 25道难度递增的标准化IQ测试题目，涵盖记忆、速度、反应、专注力和逻辑等多个维度
- 实时评分系统，提供详细的智能分析报告
- 支持年龄、性别校准的个性化评分
- 测试开始前展示说明页面，告知用户：共25道难度递增的题目、从6个选项中选出唯一答案、可跳过题目稍后返回作答

#### 2.1.2 支付订阅系统

- 一次性付费：$1.98解锁完整IQ报告和证书
- 订阅服务：$28.80/月，享受专业训练课程和定期测评
- PayPal集成支付

#### 2.1.3 大脑训练模块

- 每日推荐训练计划
- 多样化HTML5益智游戏库
- 个性化训练进度跟踪

#### 2.1.4 多元评估中心

- 职业倾向测试
- 情商(EQ)评估
- 焦虑程度检测
- 其他认知能力测评

#### 2.1.5 用户管理系统

- 无密码登录机制，通过邮箱验证码登录
- 用户档案管理，保存测试历史和训练记录
- 隐私设置和用户协议管理

#### 2.1.6 政策声明模块

- 隐私政策页面：详细说明数据收集、使用和保护政策
- 用户协议页面：服务条款和使用规定
- Cookie政策页面：Cookie使用说明和管理
- 在网站页脚和用户管理区域提供便捷入口

#### 2.1.7 定价页面

- 页面内容：展示不同的订阅计划和价格选项
- 包含双周订阅计划和每月卓越计划的对比
- 显示优惠价格和限时特价信息
- 提供开始订阅按钮
- 在用户退订后重新订阅时，通过用户下拉列表访问该页面

#### 2.1.8 Admin管理页面

**模块一：数据总览**
- 总用户数统计\n- 付费用户数及付费金额汇总
- 订阅用户数及订阅金额汇总

**模块二：详细数据**
- 每日数据列表展示
- 包含PV（页面浏览量）、UV（独立访客数）、用户数
- 新增用户数、新增付费用户数、新增付费金额
- 新增订阅用户数、新增订阅金额

**模块三：用户列表**
- 支持分页显示，每页展示20个用户
- 用户信息包含：用户邮箱、姓名、付费状态
- 订阅类型、付费金额、订阅到期时间
- 用户创建时间
- 支持用户数据的查看和管理

### 2.2 页面结构

#### 2.2.1 首页(Homepage)

- 深色头部导航(#303140)，白色主标题“IQ Train”，橙色强调色(#F78739)
- 渐变英雄区域，主打标语“Discover Your Accurate IQ”(IQ为橙色)，橙色START NOW按钮
- 主标题下方展示框：“本月进行了36,168次智商测试”|“平均智商分数为112”
- 按钮下方文字：“三分钟即可完成测试”
- 背景添加IQ相关标志性图片随机分布
- 智商分布区域：显示平均人口智商钟形曲线图表，配“立即测试你的智商！”按钮
- 名人智商轮播：史蒂芬·霍金140、埃隆·马斯克160、戴安娜王妃110、玛丽莲·梦露100
- 客户评价展示区：标题“客户心声”，3张卡片轮播显示用户评价（蓝色、橙色、紫色背景）
- 国家智商数据网格：标题“各国平均智商”，20个国家国旗和随机智商分数

#### 2.2.2 登录页面(Login Page)

- 支持邮箱+密码登录或邮箱验证码验证
- 未注册用户直接跳转至IQ测试页面

#### 2.2.3 IQ测试页面(Test Page)

- 统一顶部导航设计
- 正计时器显示(MM:SS格式)
- 两栏布局：左侧300×300问题图片区域，右侧3x2网格答案选项
- 导航控件：Previous/1-25/Next，第25题显示Submit
- 测试完成后显示结果获取界面
- 分析页面：蓝色logo(#3b82f6)，闪烁进度条(30s)，5项能力选择(Memory/Speed/Reaction/Concentration/Logic)，用户信息收集表单
- 进度提示弹窗(25%/50%/75%)

#### 2.2.4 仪表盘(Dashboard)

- 蓝色头部导航(#228CE9)
- 三大功能标签：Training | Tests | Profile，在页眉上居中展示

**Training导航：**
- Today\\\'s Train Recommend模块：每天从数据库随机展示3个HTML5游戏
- All Games模块：按游戏类型分类展示（Puzzles、Number Games等），通过iframe嵌套展示
- 游戏数据存储在专门的Html5游戏数据库表格中

**Tests导航：**
- More Tests内容：职业匹配测试、情商测试、焦虑水平测试等多元评估
- 已完成测试内容：展示用户已完成的各类测试记录，点击Results可查看详细结果
- 测试内容存储在两张数据库表中：一张存储所有测试信息，另一张存储对应题库

**Profile导航：**
- 用户账户基础信息
- 重设密码功能
- 隐私设置：隐私政策、用户协议、Cookie政策
- 退订按钮
- 登出功能
- 位于页眉最左侧，点击后下拉显示各项用户管理功能

#### 2.2.5 结果页面(Result Page)

- 参考用户提供的myiq.com_zh_checkout.png界面设计
- 展示用户IQ分数和详细分析报告
- 提供付费解锁完整报告的选项
- 显示优惠信息和付款按钮

#### 2.2.6 政策声明页面

**隐私政策页面：**
- 数据收集范围和目的
- 数据使用和分享政策
- 数据安全保护措施
- 用户权利说明（访问、修改、删除数据等）
- 联系方式

**用户协议页面：**
- 服务条款和条件
- 订阅和服务规定
- 内容使用规范
- 知识产权声明
- 解约和退款政策

**Cookie政策页面：**
- Cookie使用说明
- 必要Cookie和非必要Cookie区分
- 如何管理和禁用Cookie
- 更新通知政策

- 在网站页脚和用户管理区域提供便捷入口链接

#### 2.2.7 定价页面(Pricing)

- 页面布局：参考屏幕截图2025.12.14.png
- 展示双周订阅计划和每月卓越计划的对比
- 显示优惠价格和限时特价信息
- 提供开始订阅按钮
- 在用户退订后重新订阅时，通过用户下拉列表访问该页面

#### 2.2.8 Admin管理页面

- 数据总览模块：展示关键业务指标
- 详细数据模块：按日维度的数据统计
- 用户列表模块：分页展示用户详细信息
- 支持用户数据的筛选和管理

## 2.3 数据库设计

#### 2.3.1 数据表结构

- Question Table：存储25道IQ测试题目，每题包含6个选项
- Test Result Table：关联用户邮箱，存储测试答案和用户信息
- User Profile Table：记录用户基本信息、测试历史和订阅状态
- Html5 Game Table：存储HTML5游戏链接和分类信息
- Test Category Table：存储各类测试信息
- Test Question Table：存储各测试类型的题目库
- Policy Documents Table：存储隐私政策、用户协议、Cookie政策的文本内容
- Admin Data Table：存储业务统计数据和用户详细信息

## 2.4 测试页面交互优化

**计时器：** 正计时，以秒为单位，以“MM : SS”的格式显示经过的时间

**问题区域（桌面上的两栏布局）：**

1. **左：问题图片**
- 大型方形容器（300x300像素）
- 灰色边框，内嵌阴影
- 显示SVG格式的问题图像

2. **右侧：答案选项**
- 标题：“选择您的答案：”
- 3x2个答案框的网格
- 每个答案框：
- 白色背景，灰色边框
- 左上角的数字徽章（1-6）
- 包含答案图片
- 悬停时：橙色边框
- 选中状态：橙色边框，浅橙色背景

**导航栏（卡片底部）：**
- 灰色圆形容器
- 三列网格：
1. 带左箭头的“上一页”按钮（Q1上禁用）
2. 进度指示器：橙色药丸按钮中的“1 / 25”
3. 带右箭头的“关注”按钮（在问题25上显示为“提交”）

**完成模态：**
- 覆盖以居中的白色卡片
- "🎉 测试完成！"标题
- 显示：已回答问题数、耗时和“查看结果”橙色按钮

**交互作用：**
- 点击答案后400毫秒自动跳至下一个问题
- 键盘快捷键：1-6键选择答案，箭头键用于导航
- 计时器持续正计时。以秒为单位，以MM:SS格式进行输出
- 每个问题都会更新进度条

## 3. 技术实现

### 3.1 前端技术

- 使用Bubble平台构建
- HTML5游戏集成，通过iframe嵌套展示
- 流程自动化：测试完成自动导向支付页面
- 数据验证：确保用户信息完整性和有效性
- 政策页面静态内容展示，支持中英双语切换
- Admin页面数据可视化和用户管理功能

### 3.2 后端功能

- 自动账户创建：基于用户提供邮箱地址
- 数据持久化：测试答案和用户信息与邮箱绑定
- 权限控制：未支付用户限制访问付费内容
- 政策文档存储和管理
- 业务数据统计和用户数据分析
- Admin后台数据接口支持

### 3.3 支付集成

- PayPal一站式支付解决方案
- 一次性费用和订阅服务双重模式
- 支付状态实时更新用户权限
- 订阅用户数据同步更新

## 4. 设计风格

### 4.1 配色方案

- 主色调：深蓝灰(#303140)和科技蓝(#228CE9)
- 强调色：活力橙(#F78739)
- 辅助色：专业蓝(#3b82f6)
- 卡片背景：纯白，营造专业可信的测试环境
- Admin页面采用深色主题，主色调为深蓝灰(#303140)

### 4.2 视觉细节

- 圆角设计：统一8px圆角，营造现代感
- 卡片阴影：轻微投影效果增强层次
- 按钮样式：渐变填充配合悬停动画
- 进度条：线性进度配合颜色变化反馈\n- Admin页面采用表格和卡片式布局，数据展示清晰直观

### 4.3 交互体验

- 流程式引导：清晰的测试和支付流程
- 实时反馈：答题进度和时间提醒
- 移动适配：响应式设计确保流畅体验
- 负载优化：测试数据前端缓存，提交时同步后台
- 政策页面简洁易读的排版设计
- Admin页面支持数据导出和筛选功能
